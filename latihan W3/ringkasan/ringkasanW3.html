1. Position

Static: posisi default elemen, mengikuti alur dokumen.

Static

Relative: elemen bisa digeser dari posisi normalnya menggunakan top/left.

Relative

Absolute: posisinya mengikuti parent yang relative. Jika tidak ada, mengacu ke dokumen.

Absolute

Fixed: menempel pada viewport, tidak ikut scroll. (Lihat bar ungu di atas)

Fixed Navbar
hanya untuk demonstrasi tambahan posisi fixed

2. Z-index

Semakin besar nilai z-index, semakin depan posisinya.

z:1
z:5
z:10

3. Hyperlink

Berbagai state link:

  • a:link → link normal (belum pernah dikunjungi)
  • a:visited → link yang sudah dikunjungi
  • a:hover → saat diarahkan kursor
  • a:active → saat diklik (ditekan)

Contoh (Status `Visited` bergantung pada riwayat browser Anda):

Link Normal (a:link) Link Visited (a:visited) Hover Me (a:hover) Active (a:active)

Link juga bisa dibuat seperti tombol:
