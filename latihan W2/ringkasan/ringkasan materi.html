INI ADALAH RANGKUMAN TANPA CSS

Cara Kerja CSS

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak halaman web. CSS bekerja dengan cara menghubungkan aturan gaya (style rules) ke elemen-elemen HTML pada halaman web.

Aturan Gaya CSS

Aturan gaya CSS terdiri dari dua bagian utama: selektor (selector) dan deklarasi (declaration).
Selektor digunakan untuk memilih elemen HTML yang akan diberi gaya, sedangkan deklarasi berisi properti (property) dan nilai (value) yang menentukan bagaimana elemen tersebut akan ditampilkan.

Jenis - jenis Selector

  • Selector Tag: Memilih elemen berdasarkan nama tag HTML (misalnya, p, h1, div).
  • Selector Class: Memilih elemen berdasarkan kelas yang ditentukan dalam atribut class (misalnya, .p1).
  • Selector ID: Memilih elemen berdasarkan ID yang ditentukan dalam atribut id (misalnya, #div1).

Contoh Penggunaan CSS

Berikut adalah contoh penggunaan Selector CSS:

            p {
                color: blue; /* Selector Tag */
                font-size: 16px;
            }
            .p1 {
                color: red; /* Selector Class */
                font-weight: bold;
            }
            #div1 {
                background-color: lightgray; /* Selector ID */
                padding: 10px;
            }

Dalam contoh di atas, aturan gaya pertama akan mengubah warna teks dan ukuran font untuk semua elemen <p> di halaman web. Aturan gaya kedua akan mengubah warna teks dan membuat teks menjadi tebal untuk elemen dengan kelas "p1". Aturan gaya ketiga akan mengatur latar belakang dan padding untuk elemen dengan ID "div1".

Perbedaan ID dan Class Pada Selector

Perbedaan utama antara selector ID dan selector Class adalah bahwa ID harus unik dalam sebuah halaman web, artinya hanya boleh ada satu elemen dengan ID tertentu. Sedangkan Class dapat digunakan pada beberapa elemen yang berbeda. Oleh karena itu, selector ID biasanya digunakan untuk elemen yang memiliki peran khusus atau unik dalam halaman web, sedangkan selector Class digunakan untuk mengelompokkan elemen-elemen yang memiliki gaya yang sama.

Pewarisan pada CSS

Pewarisan (inheritance) adalah konsep di mana properti CSS yang diterapkan pada elemen induk (parent element) dapat diwariskan oleh elemen anak (child element) di dalamnya. Ini memungkinkan elemen-elemen anak untuk mewarisi gaya dari elemen induk tanpa perlu mendefinisikan ulang properti tersebut.

Contoh Pewarisan

Misalnya, jika kita memiliki aturan gaya berikut:

            body {
                font-family: Arial, sans-serif;
                color: black;
            }
            p {
                font-size: 16px;
            }

Dalam contoh di atas, elemen <body> memiliki properti font-family dan color yang akan diwariskan oleh semua elemen di dalamnya, termasuk elemen <p>. Oleh karena itu, semua teks dalam elemen <p> akan menggunakan font Arial dan berwarna hitam, kecuali jika ada aturan gaya lain yang secara eksplisit mengubah properti tersebut pada elemen <p>.

Penempatan CSS

Ada tiga cara utama untuk menambahkan CSS ke dalam dokumen HTML:

  1. Inline CSS: Menambahkan gaya langsung pada elemen HTML menggunakan atribut style. Contoh: 
    <p style="color: blue; font-size: 16px;">Ini adalah paragraf dengan inline CSS.</p>
  2. Internal CSS: Menambahkan gaya di dalam elemen <style> di dalam bagian <head> dokumen HTML. Contoh: 
                        <head>
                        <style>
                            p {
                                color: red;
                                font-size: 18px;
                            }  
                        </style>
                    </head>
  3. External CSS: Menghubungkan file CSS eksternal ke dokumen HTML menggunakan elemen <link>. Contoh: 
                        <head>
                        <link rel="stylesheet" href="styles.css">
                    </head>
    Di mana "styles.css" adalah file CSS eksternal yang berisi aturan gaya.
